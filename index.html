guidatematica — classe LaTeX per le guide tematiche del GuIT

View the Project on GitHub GuITeX/guidatematica

guidatematica is a LaTeX class used to style the handbooks written by the members of GuIT, the Italian Group of TeX/LaTeX users.

How to generate the relevant files

In order to be able to use guidatematica you need

  • Generate the class file (make cls)
  • Install it (make install)

The the makefile target 'doc' generates the documentation for the style class itself.

You are now ready to play with it!

Enjoy!!!
